# games-golang

A collection of 2D games written in Go using [Ebitengine](https://ebitengine.org/).

## Requirements

- [Go](https://go.dev/dl/) 1.21 or later
- Platform-specific Ebitengine dependencies — see the [Ebitengine install guide](https://ebitengine.org/en/documents/install.html)

## Running a Game

Each game lives in its own directory. To run one:

```bash
cd <game-folder>
go run .
```
